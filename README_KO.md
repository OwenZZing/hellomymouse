# 🔬 Hypothesis Maker — 사용 설명서 (한국어)

> 연구실 논문 PDF → AI 분석 → Research Starter Kit (.docx) 자동 생성  
> Made by @hellomymouse · kby930@gmail.com

---

## 이게 뭔가요?

연구실에 처음 들어간 대학원생을 위한 도구입니다.  
연구실 논문 PDF를 폴더에 넣으면, AI가 자동으로 분석하여 아래 내용이 담긴 Word 문서를 만들어줍니다:

- 연구실에서 진행 중인 프로젝트 목록
- 연구실이 보유한 기술/장비 목록 (한 줄 설명 포함)
- 논문별 요약 (핵심 결과 + 한계점)
- 가설 후보 H1~H7 (안전한 가설 6개 + 트렌디한 가설 1개)
- 각 가설의 별점 임팩트 / 예상 기간 / 필요 자원
- 소요 비용 추정 (KRW 기준)
- PI 확인 체크리스트
- 배경 지식 가이드 (개념, 키워드, 추천 저널)
- 첫 3개월 Roadmap

## 설치 방법

### 방법 A: exe 파일 (권장, 비개발자)

`HypothesisMaker.exe` 파일을 더블클릭하면 바로 실행됩니다.  
별도 설치 필요 없습니다.

### 방법 B: Python으로 직접 실행 (개발자)

```bash
# Python 3.11 이상 필요
cd Hypothesis-Finder
python -m venv venv
source venv/Scripts/activate        # Windows
pip install -r requirements.txt
python main.py
```

## 사용 방법

### Step 1 — 논문 준비
연구실의 최근 5년 논문 PDF를 최대한 모아 하나의 폴더에 넣습니다.  
**10~20편** 정도가 적당합니다. 너무 적으면 분석 품질이 낮아집니다.

### Step 2 — API Key 입력
프로그램 상단에서 사용할 AI를 선택하고 API Key를 입력합니다.

| AI | API Key 발급 |
|----|------------|
| **Claude (추천)** | [console.anthropic.com](https://console.anthropic.com) → API Keys |
| **OpenAI (GPT)** | [platform.openai.com](https://platform.openai.com) → API keys |
| **Google Gemini** | [aistudio.google.com](https://aistudio.google.com) → Get API key |

> API Key는 프로그램 설정 파일에 저장됩니다 (`~/.hypothesis_maker_config.json`).  
> 공용 컴퓨터에서 사용 후에는 Key를 삭제하세요.

### Step 3 — 논문 폴더 선택
"연구실 논문 폴더" 옆 **찾기** 버튼으로 폴더를 선택합니다.

### Step 4 — (선택) 추가 참조 논문
교수님이 "이 논문도 봐"라고 추천하신 논문이 있다면, 별도 폴더에 넣고 "추가 참조 논문 폴더"에서 선택합니다.  
이 논문들은 연구실 논문과 구분하여 참조 문헌으로 분석됩니다.

### Step 5 — 프로젝트 목록 파악 (Stage 0)
**"🔍 프로젝트 목록 파악"** 버튼을 클릭합니다.  
AI가 논문 제목과 초록만 빠르게 읽어 연구실의 프로젝트 목록을 추출합니다. (30초~1분)

### Step 6 — 배정된 프로젝트 선택
- 교수님이 특정 프로젝트를 배정하셨다면 **"목록에서 선택"**으로 선택하거나 직접 입력하세요.
- 배정이 없다면 비워두세요 → 모든 프로젝트를 고르게 분석합니다.

**배정된 프로젝트가 있는 경우:**
- H1~H4: 해당 프로젝트에 집중된 가설
- H5~H7: 나머지 프로젝트의 사이드 가설

**배정이 없는 경우:**
- H1~H7: 연구실 전체 프로젝트에서 고르게 가설 도출

### Step 7 — (선택) 교수님 지시사항 입력
교수님이 추가로 말씀하신 내용을 자유롭게 입력합니다.  
예시: `"reinforcement learning 방향으로 보면 좋겠다"`, `"실제 로봇 실험까지 가는 논문 목표"`

### Step 8 — 분석 시작
**"▶ 분석 시작"** 버튼을 클릭합니다.  
논문 수에 따라 3~10분 소요됩니다. 진행 상황이 아래 로그에 표시됩니다.

> ⚠ **API 비용 안내**  
> - Claude Sonnet: 논문 10편 기준 약 $0.3~0.8 (₩400~1,100)  
> - GPT-4o: 논문 10편 기준 약 $0.5~1.5 (₩700~2,000)  
> - Gemini Flash: 논문 10편 기준 약 $0.05~0.2 (₩70~280)

### Step 9 — 저장 및 열기
분석이 완료되면 저장 위치를 선택하고 **"📄 리포트 생성"**을 클릭합니다.  
생성된 `.docx` 파일은 Word에서 열어 확인하세요.

## 가설 임팩트 별점 가이드

| 별점 | 의미 |
|-----|-----|
| ★☆☆☆☆ | 졸업 아슬아슬. 심사위원에 많이 깎임 |
| ★★☆☆☆ | 졸업 무난. 취직은 다른 스펙이 받쳐줘야 함 |
| ★★★☆☆ | 졸업 안정적. 취직 시 논문으로 설명 가능 |
| ★★★★☆ | 졸업 오래 걸릴 수 있음. 끝나면 포지션 골라감 |
| ★★★★★ | 분야를 바꾸는 논문. 5년이 넘어도 worth it |

> ※ 예상 기간은 해당 기술을 이미 숙지한 경우 기준입니다.  
> 신기술 학습 곡선을 포함하면 실제로는 **2~3배** 소요될 수 있습니다.
